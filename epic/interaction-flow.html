Interaction Flow

Title: Streaming on Twitch
Description: A registered user wants to start a stream on their account.
Name and Role of Persona: Joesph Biggs, registered user
Preconditions: An account on Twitch, equipment needed to stream, and applications on his computer to stream with.
Postconditions: Registered user is streaming on his account.

Interaction Flow:

  • User selects their account.
  • The server redirects user to their url.
  • User launches streaming service application from their desktop.
  • The server gets a connection from the streaming application and connects to their stream.
  • User selects "begin stream" from their streaming application.
  • The server begins hosting the stream. The server sets the user's status to live.
